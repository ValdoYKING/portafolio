ConferExpo

ConferExpo es una plataforma web diseñada para la gestión eficiente de eventos y conferencias. Utiliza Flask para el backend y MongoDB como base de datos, lo que garantiza escalabilidad y rendimiento.

Características Principales

  • Gestión de registros y asistencia
  • Creación de horarios personalizados
  • Interfaz de usuario intuitiva y responsiva
  • Integración con plataformas de pago
