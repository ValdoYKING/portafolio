Incognito

Incognito es una plataforma web desarrollada para la evaluación y gestión de servicios. Implementada con Node.js y MongoDB, permite manejar datos dinámicos y plantillas personalizadas para las vistas.

Características Principales

  • Evaluación de servicios en tiempo real
  • Reportes detallados y análisis de datos
  • Dashboard de administración intuitivo
  • Seguridad y privacidad de datos
